# भजन: अमरपुर ले चलो सजना (कबीर साहब)

- अमरपुर ले चलो सजना,\
  अमरपुर ले चलो सजना।।

- अमरपुर की साँकर गलियाँ,\
  अड़बड़ है चलना।।

- गुरु ज्ञान शब्द की ठोकर,\
  उघर गये झपना।।

- वहीं अमरपुर लागि बज़रिया,\
  सौदा है करना।।

- अमरपुर संत बसत है वहीं,\
  दर्शन है लहना।।

- संत समाज जहाँ बैठी,\
  वहीं पुरुष अपना।।

- कहत कबीर सुनो भाई साधो,\
  भवसागर तरना।।

**~ गुरु कबीर साहब**
